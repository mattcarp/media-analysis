Features

No Frameworks

This example uses no front end frameworks (such as extjs or angular). On the back end, only a configurable binary executable listens on a port, and redirects the upload chunks to an AWS S3 bucket.

Multi-Threaded

Uploads are in the multi-part format, which allows large files to be segmented into chunks that are uploaded simultaneously.

Resumable

A user can close the browser, reboot the machine, etc., and the page will auto re-try the upload when they return.

Pausable

A user can pause an upload (for example, if they are about to go offline), and resume later.
